## Суть проблемы
- **Текущая ситуация:**
  - Создатели контента (видеоблогеры, студии) сталкиваются с трудностями при создании мультиязычного контента.
  - Подростки хотят экспериментировать с голосами и создавать уникальный контент, но у них ограничен бюджет.
  - Малые студии не могут позволить себе дорогую профессиональную озвучку.
- **Основная проблема:**
  - Отсутствие доступного инструмента для переозвучки видео с качественным липсинком.
  - Высокая стоимость профессиональной озвучки (~3,000 ₽/минута).
  - Низкая доступность технологий для подростковой аудитории.

- **Дефициты конкурентов:**
  - Нет возможности использовать свой голос или выбирать из готовых голосов для реальных лиц.
  - Ограниченный выбор языков для перевода.
  - Высокая цена для массового потребителя.

## Почему мы уникальны
- Мы предлагаем комплексное решение: переозвучка + перевод + липсинк.
- Качественный липсинк для реальных лиц.
- Базовый тариф доступен всем, включая подростков.

## Какие технологии используем
- **TTS (Text-to-Speech):** FastSpeech или Tacotron 2.
- **Липсинк:** VideoFace2Face.
- **API перевода:** Яндекс.Translate API.

## Какие метрики будем отслеживать
- **DAU/MAU:** Число активных пользователей.
- **Retention Rate:** Удержание пользователей.
- **Conversion to paid:** Конверсия бесплатной версии в платную.
- **CAC/LTV:** Стоимость привлечения vs lifetime value пользователя.

## Когда достигнем PMF
- Через 3 месяца после запуска MVP, когда количество активных пользователей достигнет 200+.

## Главная страница
- **Описание:**
  - **Название:** ReVoice.
  - **Слоган:** «Создай свой голосовой мир».
  - **Ключевые элементы:**
    - Кнопка регистрации/входа.
    - Короткий видеоролик о работе платформы.
    - Call-to-action: «Попробуй бесплатно».

## Страница инструментария
- **Описание:**
  - Загрузка видео.
  - Выбор языка для перевода.
  - Выбор голоса (свой или заготовленные).
  - Настройка эмоционального спектра голоса (для платных пользователей).
  - Выбор фрагмента для переозвучки.

## Страница личных роликов
- **Описание:**
  - Личный профиль с загруженными роликами.
  - Возможность публикации и скачивания.
  - Лимиты для бесплатных и платных пользователей:
    - Бесплатный тариф: 25 попыток по 5 минут/день.
    - Премиум тариф: 3 попытки полнометражного кино или 45 коротких фрагментов.

## Страница личной анкеты
- **Описание:**
  - Тестирование голоса при регистрации:
    - Пользователь записывает текст с всеми гласными и согласными буквами.
    - Текст становится частью портфолио голоса.
  - Выбор роли на площадке:
    - Артист (переозвучивает видео).
    - Создатель (загружает свои видео для озвучки).
    - Любитель (с указанием предпочтений).
  - Демонстрация голоса:
    - Пользователи могут прослушать ваш голос и оценить его.
  - Чат для списки с остальными пользователями:
    - Пользователь выбирает по нику пользователя и пишет желаемое письмо.
    - Входящие сообщения с указанием отправившего пользователя и указанием его роли.

## Тарифы
1. **Бесплатный тариф:**
   - Возможность использовать свой голос или 3 стандартных.
   - Перевод на 3 языка.
   - 25 попыток по 5 минут/день.
   - Реферальная система: приглашайте друзей и получайте +30 дней премиум-доступа.
2. **Стандартный тариф (990 ₽/мес.):**
   - Переозвучка полнометражных фильмов.
   - Доступ к большей библиотеке голосов (10+).
   - Перевод на дополнительные языки.
   - 3 попытки полнометражного кино или 45 коротких фрагментов.
3. **Профессиональный тариф (2,990 ₽/мес.):**
   - Полная настройка эмоционального спектра голоса.
   - Выбор ролей для каждого голоса.
   - Безлимитный трафик.
   - Доступ к эксклюзивным голосам.
